# dotfiles
> personal dotfiles to set up a new computer(so called instance)

### prerequisites
- python3 pip3
- curl
- apt-get or brew

### targets(all the items are optional)
- git
- tools
    - ag
    - autojump
    - powerline
- node
- vim
- zsh
- tmux
- env
- miscellaneous
    - ideavim
    - ssh
